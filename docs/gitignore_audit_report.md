# .gitignore Audit Report — NeedNow AI

## Summary

Production-ready `.gitignore` created at project root. Verified that no secrets, caches, or generated artifacts will be committed.

| Category | Patterns | Rationale |

| Next.js Build | `.next/`, `out/` | Generated build output |

| FAISS Indexes | `backend/faiss_indexes/`, `*.index` | Generated ML artifacts |
| ChromaDB | `chroma_db/`, `*.sqlite3` | Generated vector store |
| Logs | `*.log`, `logs/` | Runtime logs |
| OS Files | `.DS_Store`, `Thumbs.db` | OS-specific metadata |
| IDE | `.vscode/`, `.idea/` | Editor configuration |
| Coverage | `.coverage`, `htmlcov/`, `coverage.xml` | Test coverage output |
| Build | `dist/`, `build/` | Production builds |
